# Bibliotec
<br>

<p>O <strong>Bibliotec</strong> é um sistema de biblioteca desenvolvido na ETEC Aristóteles Ferreira, com o objetivo de facilitar o gerenciamento e facilitar o empréstimo e devolução dos livros.</p>
